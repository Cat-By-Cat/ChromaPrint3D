# 任务手册索引

任务手册用于把高频改动沉淀成固定流程，避免漏改和漏测。

## 使用方式

1. 先选择最接近的任务手册
2. 按“影响文件 -> 实施步骤 -> 风险点 -> 回归检查”执行
3. 完成后回填文档同步项
