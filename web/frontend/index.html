ChromaPrint3D - 图片转多色3D打印模型

ChromaPrint3D 是一款免费的在线工具，可以将图片转换为多色 3D 打印 3MF 模型。

核心功能

  • 图片转多色 3MF 模型 - 支持 JPG、PNG、BMP、TIFF、SVG 格式
  • Bambu Studio 预设自动写入 - 0.2mm/0.4mm 喷嘴，观赏面朝上/朝下
  • 颜色配方智能匹配 - 自动将模型颜色匹配到最接近的耗材丝槽位
  • 1-20 层自由调色 - 不再受限于固定色层数
  • 自定义配方编辑器 - 手动设计配色方案并实时预览
  • 校准板生成 - 构建和管理 ColorDB 颜色数据库
  • 图像预处理 - SVG 矢量化、背景抠图

使用方式

  • 在线使用（推荐）
  • 下载桌面版（Windows / macOS / Linux）
  • Docker 部署：docker run -d -p 8080:8080 neroued/chromaprint3d:latest
